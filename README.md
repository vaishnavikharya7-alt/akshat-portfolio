# Akshat Gupta - 3D Portfolio Website

A modern, interactive 3D portfolio website for Akshat Gupta, a Full Stack Engineer & AI Enthusiast.

## ✨ Features

- 🎨 **3D Animated Background** - Built with Three.js
- ⌨️ **Typing Animation** - Dynamic text effects
- 📊 **Animated Stats Counter** - Numbers that count up
- 💎 **Glassmorphism Design** - Modern card effects
- 📱 **Fully Responsive** - Works on all devices

## 🛠️ Tech Stack

- HTML5 / CSS3
- JavaScript (ES6+)
- Three.js
- AOS (Scroll Animations)
- Font Awesome Icons
